Todo: Documentation for AlertPlus

In order to use AlertPlus, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "AlertPlus" );

Then you can create an instance of the plugin object when you need it like this:

 ap = app.CreateAlertPlus();


Global Variables:

Dialog Themes:

Light: ap.THEME_LIGHT (value: 3)

Dark: ap.THEME_DARK (value: 2)

Basic: ap.THEME_BASIC (value: 1)

Popup Length:

Long: ap.LENGTH_LONG (value: 1)

Short: ap.LENGTH_SHORT (value: 0)
